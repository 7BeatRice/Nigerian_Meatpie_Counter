# Nigerian Meatpie Counter

Created by: Beatrice Olaosebikan

This web app: Uses React states and hooks to create a meatpie counter, similar to the popular cookie counter.

Time spent: 3 hours spent in total

## App Features



- [x] **The app displays a Pastry, and the three upgrade options**
  - [x] Title , count, and click multiplier is displayed 
  - [x] Three different types of upgrades are featured
  - [x] Each Upgrade include a title, description of its affect, and its cost
  - [x] The cost is displayed as a button
  
- [x] **Count, cost and click multiplier updates accordingly**
  - [x] Clicking on an upgrade reduce the count by the cost of the upgrade
  - [x] Clicking on an upgrade should increase the click multiplier accordingly and when clicked the count should go up accordingly
  - [x] Clicking on an upgrade should increase the cost of the item
